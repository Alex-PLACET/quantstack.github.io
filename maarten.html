Maarten Breddels
Scientific software developer

@maartenbreddels

Maarten Breddels is a developer and data scientist working mostly with Python, C++ and Javascript in the Jupyter ecosystem. His expertise ranges from fast numerical computation, API design, to 3-D visualization. Maarten holds a Bachelor in ICT, a Master and PhD in Astronomy and likes to code and solve problems.

Maarten is the main author of ipyvolume and vaex and is a core contributor to the Jupyter interactive widgets.
